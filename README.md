# 🍲 NomNom UI Library — Documentation

Welcome to **NomNom UI**, an ultra-minimalist, elegant, and full-premium user interface library built for Roblox. Engineered as a modern, clean alternative to traditional bulky interfaces, NomNom features highly optimized fluid animations, an automated premium intro loading screen, asset customization via Image Asset ID, and a sleek **Dynamic Island** transition state when minimized.

---

## 🚀 Getting Started (Booting the Library)

To initialize the library and create your main hub window, paste the following boilerplate code into your executor:

```lua
local NomNom = loadstring(game:HttpGet("[https://raw.githubusercontent.com/1MN00X/NomNom/refs/heads/main/source.lua](https://raw.githubusercontent.com/1MN00X/NomNom/refs/heads/main/source.lua)"))()

local Window = NomNom.new({
    Title = "NomNom Premium Hub 🍲",
    LogoId = "rbxassetid://0", -- Replace with your Roblox Asset ID (e.g., "rbxassetid://123456")
    ThemeColor = Color3.fromRGB(138, 43, 226) -- Premium Accent Color (RGB)
})

```

## 🛠️ UI Components & Elements

NomNom provides clean, reactive elements to bind your scripts. Below is the complete API guide on how to add and configure each component manually.

### 1. 🖱️ Click Button (Single-Click Action)

Used for actions that trigger instantly upon a single click (e.g., UI destruction, opening secondary menus, or instant utility resets).

* **Syntax:** `Window:CreateButton(name, callback)`
* **Code Example:**

```lua
Window:CreateButton("Optimize Game FPS", function()
    -- Your custom script execution goes here
    print("FPS Optimization injected successfully!")
end)

```

---

### 2. 🎛️ Toggle Switch (On / Off State)

Used for continuous features that require an active (true) or inactive (false) state. The UI automatically handles the smooth sliding animation and background color shifts dynamically.

* **Syntax:** `Window:CreateToggle(name, default_state, callback)`
* **Code Example:**

```lua
Window:CreateToggle("Sprint Booster", false, function(state)
    if state then
        -- Action when turned ON
        game.Players.LocalPlayer.Character.Humanoid.WalkSpeed = 50
        print("Sprint Booster: Enabled")
    else
        -- Action when turned OFF
        game.Players.LocalPlayer.Character.Humanoid.WalkSpeed = 16
        print("Sprint Booster: Disabled")
    end
end)

```

## 🌴 The Dynamic Island Feature

NomNom removes traditional, cluttering minimize configurations.

* Clicking the **"—"** button on the top right will automatically trigger a smooth transition, compressing the entire GUI into a sleek, pill-shaped **Dynamic Island** at the top center of the player's screen.
* To restore the menu back to its full size, the user simply clicks directly on the Dynamic Island pill.

*Note: The hover feedback, interactive scaling, and transition physics are handled natively inside the source code; no secondary listener setup is required in your script.*

## 🎨 Advanced Customization Tips

### Custom Asset Images (Logos)

To display a logo inside your premium header:

1. Upload your icon/image to Roblox as a Decal.
2. Copy the **Asset ID** generated by Roblox.
3. Replace `"rbxassetid://0"` inside the configuration with your valid ID (e.g., `"rbxassetid://987654321"`). Leave it as `0` if you prefer a completely clean, text-only layout.
